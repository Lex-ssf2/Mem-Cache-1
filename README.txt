Para hacer el uso del programa es de la siguiente manera
Lees el archivo de la cache como una entrada es decir

./compilado < ./entrada.txt

el archivo entrada.txt debe estar compuesto de la siguiente manera

n (tipo de memoria, 1 = correspondencia directa, 2 = asociativa por conjuntos, 3 = completamente asociativa)

i j k (i = el tamano del bloque,j = el numero de palabras, k = el numero de vias)

direcciones de memoria separadas por un espacio
